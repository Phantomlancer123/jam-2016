## Rules

- Maximum life: 3

## Getting Started
1. Set path (e.g. `cd C:\xampp\htdocs\jojoee.com\jam-2016`)
2. Install global - `npm install -g bower gulp browser-sync`
3. Install dependencies - `bower install && npm install`
